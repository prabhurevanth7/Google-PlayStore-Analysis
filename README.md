# Google Play Store Data Analysis

## Project Overview
Exploratory Data Analysis on Google Play Store dataset to answer:
**Which app categories have the highest ratings?**

## Dataset
- Source: Kaggle Google Play Store Dataset
- Size: 10,841 apps across 33 categories
- Columns: App, Category, Rating, Reviews, Size, Installs, Type, Price, etc.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- EVENTS, EDUCATION, and ART_AND_DESIGN are the top 3 highest rated categories (avg above 4.35)
- DATING has the lowest average rating (3.97) due to polarizing user experiences
- All 33 categories have 42+ apps, making the results statistically reliable

## Visualizations
- Top 5 categories by average rating (horizontal bar chart)
- Bottom 5 categories by average rating (horizontal bar chart)
